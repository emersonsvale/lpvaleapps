# PP Neue Montreal Font Files

## Instruções para adicionar os arquivos de fonte

1. Baixe a fonte PP Neue Montreal do site: https://www.cufonfonts.com/font/pp-neue-montreal
2. Extraia os arquivos de fonte do ZIP baixado
3. Copie os seguintes arquivos para a pasta `public/fonts/`:

### Arquivos necessários:

- `PPNeueMontreal-Book.woff2` (ou .woff)
- `PPNeueMontreal-BookItalic.woff2` (ou .woff)
- `PPNeueMontreal-Thin.woff2` (ou .woff)
- `PPNeueMontreal-Medium.woff2` (ou .woff)
- `PPNeueMontreal-SemiBoldItalic.woff2` (ou .woff)
- `PPNeueMontreal-Bold.woff2` (ou .woff)

### Nota sobre licença:

A fonte PP Neue Montreal é "Free for personal use". Para uso comercial, é necessário entrar em contato com a Pangram Pangram Foundry: https://pangrampangram.com/

### Conversão de formato (se necessário):

Se você tiver apenas arquivos .ttf ou .otf, pode converter para .woff2 usando ferramentas online como:
- https://cloudconvert.com/ttf-to-woff2
- https://convertio.co/ttf-woff2/
